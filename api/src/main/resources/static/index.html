Buzzworthy Resource Locator (BRL) — REST API Operations:

User documentation available @ http://localhost:8080/BRL/api/

GET http://localhost:8080/BRL/api/wiki/resource

Purpose — Retrieve all resource records from the database.

Request header — Bearer: abcdefg...tuvwxyz where this string is an SSO token.

Return body — {{resource},{resource},...,{resource}} where each resource is in the format shown at the bottom of this page.

Return code — 200 (successfully retrieved), 400 (misformatted request), 401 (unable to authenticate user), 403 (insufficient user permissions), 404 (unable to retrieve resources)

POST http://localhost:8080/BRL/api/wiki/resource

Purpose — Insert a new resource record into the database.

Request header — Bearer: abcdefg...tuvwxyz where this string is an SSO token.

Request body — {resource} where the resource is in the format shown at the bottom of this page.

* Note that inserted resources should omit the "comments", "upVotes", and "reviewFlags" fields

Return body — {resource} where the resource is the inserted resource in the format shown at the bottom of this page.

Return code — 201 (successfully added), 400 (misformatted request), 401 (unable to authenticate user), 403 (insufficient user permissions), 404 (unable to add resource)

POST http://localhost:8080/BRL/api/wiki/resource/{resourceId}/comment

Purpose — Add a comment to an existing resource record.

Request header — Bearer: abcdefg...tuvwxyz where this string is an SSO token.

Request body — {comment} where the comment is in the format shown at the bottom of this page.

Return body — {resource} where the resource is the newly commented-on resource in the format shown at the bottom of this page.

Return code — 201 (successfully added), 400 (misformatted request), 401 (unable to authenticate user), 403 (insufficient user permissions), 404 (unable to add comment)

POST http://localhost:8080/BRL/api/wiki/resource/{resourceId}/upvote

Purpose — Add an upvote to an existing resource record.

Request header — Bearer: abcdefg...tuvwxyz where this string is an SSO token.

Request body — {upVote} where the upVote is in the format shown at the bottom of this page.

Return body — {resource} where the resource is the newly upvoted resource in the format shown at the bottom of this page.

Return code — 201 (successfully added), 400 (misformatted request), 401 (unable to authenticate user), 403 (insufficient user permissions), 404 (unable to add upVote)

POST http://localhost:8080/BRL/api/wiki/resource/{resourceId}/reviewFlag

Purpose — Add a flag for manager review to an existing resource record.

Request header — Bearer: abcdefg...tuvwxyz where this string is an SSO token.

Request body — {reviewFlag} where the reviewFlag is in the format shown at the bottom of this page.

Return body — {resource} where the resource is the newly flagged resource in the format shown at the bottom of this page.

Return code — 201 (successfully added), 400 (misformatted request), 401 (unable to authenticate user), 403 (insufficient user permissions), 404 (unable to add reviewFlag)

DELETE http://localhost:8080/BRL/api/wiki/resource/{resourceId}

Purpose — Delete a specific existing resource record.

Request header — Bearer: abcdefg...tuvwxyz where this string is an SSO token.

Return code — 200 (successfully deleted), 400 (misformatted request), 401 (unable to authenticate user), 403 (insufficient user permissions), 404 (unable to delete specified resource)

DELETE http://localhost:8080/BRL/api/wiki/resource/{resourceId}/comment/{commentId}

Purpose — Delete a specific comment from a specific existing resource record.

Request header — Bearer: abcdefg...tuvwxyz where this string is an SSO token.

Return code — 200 (successfully deleted), 400 (misformatted request), 401 (unable to authenticate user), 403 (insufficient user permissions), 404 (unable to delete specified comment)

DELETE http://localhost:8080/BRL/api/wiki/resource/{resourceId}/upvote/{upvoteId}

Purpose — Delete a specific upVote from a specific existing resource record.

Request header — Bearer: abcdefg...tuvwxyz where this string is an SSO token.

Return code — 200 (successfully deleted), 400 (misformatted request), 401 (unable to authenticate user), 403 (insufficient user permissions), 404 (unable to delete specified upVote)

DELETE http://localhost:8080/BRL/api/wiki/resource/{resourceId}/reviewFlag/{flagId}

Purpose — Delete a specific reviewFlag from a specific existing resource record.

Request header — Bearer: abcdefg...tuvwxyz where this string is an SSO token.

Return code — 200 (successfully deleted), 400 (misformatted request), 401 (unable to authenticate user), 403 (insufficient user permissions), 404 (unable to delete specified reviewFlag)
